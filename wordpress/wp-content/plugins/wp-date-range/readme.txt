Date range for Posts or Custom types

== Description ==

This plugin let you choice a date range to publish your Posts or Custom Types

== Installation ==

1. Upload folder wp-date-range to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin from settings menu

== Frequently asked questions ==

<h4>Is it possible to change the date format?</h4>
<p>Yes go to Settings in admin panel</p>

<h4>Does it work for related comment like recent comments?</h4>
<p>No, the plugin is not linked to related comments</p>

<h4>Does it work for pages?</h4>
<p>No, plugin doesn't work for pages only for Posts or Custom Types</p>

<h4>Does all date fields required?</h4>
<p>No, date range fields are optionals</p>

<h4>Does it recommanded for women as well?</h4>
<p>Yes, with moderation. It could be hard for her to choose the date format ;)</p>
